TASK 2 - Data Model

My class which represented the items in my catalogue consisted of the following attributes

  • id
  • title
  • developer
  • genre
  • relDate
  • setting

The id attribute was a single digit integer value and the primary key.

The purpose of this attribute was to be a unique identifier to be used when sending data to my template


The title attribute was a string. It was used to represent the title of the game.

This attribute could not be null as every game has some form of a title even if its not the final one


The developer attribute was a string. It was used to represent the developer of the game.

The developer attribute could not be null as every game has a developer


The genre attribute was a string. it was used to represent the genre of the game.

The genre attribute could not be null because every game has a genre


The relDate was a string. It was used to represent the release date of the game

The relDate attribute could be null as some games are yet to release or have a release date announced
